# vagrant-debian
==============

## Overview

Vagrant containing :

- Debian 7.1 (Wheezy)
- Apache 2
- PHP 5.4
- MariaDB 5.5
- memcached
- xdebug
- git
- Composer

Thanks to [Jean-fei](https://github.com/jeanfei) who created the Debian box.

Inspired by the great project [varying-vagrant-vagrants](https://github.com/10up/varying-vagrant-vagrants)
